arguments


based on instruction the argument could be:
- address == 32bit unsigned integer in little endian (e.g.: jump instruction)
- constant == 64 bit signed integer in little endian (e.g.: loadConstant instruction)
- data == 5 bit or 7 bit encoded value in little endian (e.g.: add, mov, consoleWrite instructions)

data
- has to be decoded have two types - register or memory

register
- 5 bit long
- stars with bit 0 followed with 4 bit register index
- example 00001 is (0-0001) register with index 1 (r1)

memory
- 7 bit long
- starts with bit 1 followed with two bits size 00, 01, 10, 11 followed with register index which containing address to data memory
- example 1110010 (1-11-0010) memory size of qword (64bit) with address stored under register 2 (qword[r2])
